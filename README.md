# Torn Stock Pocket

> A TamperMonkey script for one-click trading on torn.com's stock market.

  <img src="https://editor.torn.com/c98659d1-66eb-4981-8afb-cb4fe08a0e6d-3564828.gif" alt="Demo" width="712" />

## What It Does

Create customizable quick buttons for any stock. Buy all or by amount, percentage, or keep a specific cash reserve. Sell all or by amount or sell to hit a cash target. One-click buttons to execute trades. Highly reactive UI with dark & light mode support, and looks great on PDA/mobile.

## Features

**Quick Buttons** — Custom label, color, icon, and share count display per button

**Trade Blocking** — Block buys/sells globally or per-stock to prevent accidental trades

**Live Tracking** — Stock prices, shares owned, and cash update in real time; buttons recalculate automatically

**Undo / Redo** — Step back through configuration changes if you make a mistake

## Trade Modes

| Mode | Description |
|------|-------------|
| **Buy** | By amount, % of cash, or keep a cash reserve |
| **Sell** | By amount or sell until you reach a cash target |
| **All** | Buy with all cash or sell all shares — one click |

## Getting Started

If on PC, install [Tampermonkey](https://www.tampermonkey.net/) then click the install button below. If on Torn PDA simply click the install button below. Head to the stock market page, the panel appears above the stock list. Select a stock, configure buttons for it, and you're good to go.
